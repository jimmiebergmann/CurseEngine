Real-time game engine framework and editor.  
**Warning:** Under heavy development.

## Build
```sh
$ git clone --recurse-submodules https://github.com/jimmiebergmann/CurseEngine.git
$ mkdir CurseEngine/build
$ cd CurseEngine/build
$ cmake ..
```
